# 404-Time-Alert-Tasker-Termux
A small Termux script that notifies you at 4:04 AM/PM with toast, notification, and vibration. Easily customizable.


## Instructions
* Make sure you have Tasker, Termux and Termux API installed on your device.
* Import the profile in Tasker by holding the Profiles section.
* Put "404.sh" in ~/.termux/tasker/ on Termux.

## Note 
You can also customize the script and the profiles however you desire.
